This is the Arduino code to have an NTP-updated Wi-Fi clock on the MeLife ESP32-32E "CYD" ("Cheap Yellow Display"). Update Wi-Fi settings as needed for your WI-FI network, as well as desired NTP server, Timezone, Font,
colors, etc. 
There is a hidden button at the top-right on the touchscreen to temporarily displaynetwork information.

Description: ESP32 Display 2.8 inch, ESP32-32E WiFi+BT Dual-core TFT Module with Acrylic Case, ESP32 Development Board ILI9341 Driver 240X320 Smart LCD Display Screen for Arduino IOT

Arduino IDE Setup:
-Select Board: "ESP32 Dev Module"
-Port: USB COM port
-Libraries needed:
  -Adafruit BusIO by Adafruit
  -Adafruit GFX Library by Adafruit
  -Adafruit ILI9341 by Adafruit
  -Adafruit Touchscreen by Adafruit
  -TFT_eSPI by Bodmer
  -XPT2046_Touchscreen by Paul Stoffregen
  -possibly others as directed...
